# les-sandwichs-du-docteur-v1
PWA de gestion pour sandwicherie axée sur l'alimentation saine et la prévention santé. Permet de superviser l'activité : gestion des commandes, suivi des stocks, tableau de bord analytique. Conçue par un médecin généraliste pour allier plaisir gustatif et bien-être. Stack moderne, responsive et installable sur mobile.
